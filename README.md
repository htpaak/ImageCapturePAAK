# 고화질 화면 캡처 프로그램

이 프로그램은 모니터에 표시되는 화면을 고화질로 캡처하여 PNG 형식으로 저장할 수 있는 도구입니다.

## 기능

- 고화질 화면 캡처
- 직관적인 GUI 인터페이스 (PyQt5 기반)
- PNG 형식으로 이미지 저장
- 전체 화면 또는 선택 영역 캡처 지원

## 필요 사항

- Python 3.7 이상
- 의존성 라이브러리:
  - PyQt5: GUI 인터페이스
  - Pillow: 이미지 처리
  - mss: 고화질 화면 캡처
  - PyAutoGUI: 화면 제어
  - python-dotenv: 환경 변수 관리

## 설치 및 실행

1. 의존성 설치:

```
pip install -r requirements.txt
```

2. 프로그램 실행:

```
python main.py
```
